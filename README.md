# AI Workflow Homework

This repository combines a Next.js landing page with a small, structured AI workflow system for guided implementation and review.

## Structure

- `AGENTS.md` -- project rules and working constraints for AI assistance
- `workflows/` -- reusable step-by-step processes such as `spec-check.md`
- `agents/` -- agent roles and behavior definitions such as `spec-reviewer.md`
- `.agents/skills/` -- local execution skills, including the included `demo-skill`
- `tests/` -- simple validation prompts used to check workflow behavior

## Project

The `ai-project/` app is a Next.js App Router landing page for AI Architect Pilot. It is used as the working surface for applying the workflow rules, reviewing changes, and validating structured AI execution.
